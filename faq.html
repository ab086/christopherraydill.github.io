Game Related

My game is crashing!

First off, are you using mods? We have no problem with people using mods, however we will not provide any technical support for games that have mods installed. You'll have to troubleshoot if it is indeed a mod causing the problem.

If you're not using mods and/or the mod is not causing the problem, try some of these hot fixes:

  • With mods: Remove all mods and relaunch the game.
  • Run the game with administrative privlages.
  • Whitelist the game's directory with your anti-virus software.
  • Purge the cache files from your system. Don't worry, they'll be recreated when the game launches again.
  • Last resort: Do a full reinstall of the game.

If none of these seemed to help you, then please visit our contact us page and submit a ticket. Be sure to include your "log.txt" file!

When is the next game update?

As of right now, we have no set update schedule. Our team is pretty small. Small updates happen fairly quickly, but larger updates can take us a long time to publish.
